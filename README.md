# BYOK AI Desktop

A lightweight Windows desktop application for AI chatting. Bring your own API key and start chatting!

## Key Features
- **BYOK (Bring Your Own Key)**: Use your OpenAI/Gemini/Claude API keys.
- **Privacy First**: No intermediate server. Direct connection to AI providers.
- **Modern UI**: Built with Python & Flet (Flutter-based).

## How to Start
1. Clone this repo: `git clone https://github.com/your-id/your-repo-name.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python main.py`

## Security
Your API Key is only stored locally on your machine.
